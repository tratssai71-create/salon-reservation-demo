# サロン予約システム（プロトタイプ）

メニュー→日付→時間→スタッフ→お客様情報→確認→完了、というホットペッパー的な予約フローを持つ独自の予約システムです。予約が確定すると、Googleカレンダーに自動でイベントが作成され、お客様に確認メールが送信されます。

## 構成

| ファイル | 役割 |
|---|---|
| `index.html` / `style.css` / `app.js` | フロントエンド（予約フォーム本体） |
| `config.js` | 店舗名・メニュー・スタッフ・営業時間・API接続先の設定 |
| `gas/Code.gs` | バックエンド（Google Apps Script）。カレンダー登録・メール送信を行う |
| `gas/appsscript.json` | GASのマニフェスト（デプロイ設定） |

## 今すぐ試す（デモモード）

`config.js` の `API_URL` が空文字のままだと、実際のカレンダーに接続せず、ブラウザ内だけで予約フローを最後まで試せる「デモモード」で動作します。`index.html` を開くだけで確認できます。

## 本番接続（Googleカレンダーに自動追加する）

### 1. 予約用のGoogleカレンダーを用意する

- サロンで使うGoogleアカウントにログインし、Googleカレンダーで新しいカレンダーを作成（または既存のメインカレンダーを利用）
- 作成したカレンダーの「設定と共有」→「カレンダーの統合」から **カレンダーID** をコピーしておく（メインカレンダーを使う場合はコードの `CALENDAR_ID` を `"primary"` のままでOK）

### 2. Google Apps Script プロジェクトを作成する

1. [script.google.com](https://script.google.com/) にアクセスし、「新しいプロジェクト」を作成
2. デフォルトの `Code.gs` の中身を、このフォルダの `gas/Code.gs` の内容で置き換える
3. 左メニューの歯車アイコン（プロジェクトの設定）→「`appsscript.json` マニフェスト ファイルをエディタで表示する」にチェックを入れる
4. `appsscript.json` の中身を、このフォルダの `gas/appsscript.json` の内容で置き換える
5. `Code.gs` 冒頭の `CONFIG` を実際の内容に書き換える
   - `CALENDAR_ID`: 手順1で用意したカレンダーID（自分のメインカレンダーなら `"primary"`）
   - `NOTIFY_EMAIL`: 予約が入るたびに通知を受け取りたいメールアドレス（不要なら空文字）
   - `SPREADSHEET_ID`: 予約ログを記録したいスプレッドシートのID（不要なら空文字）
   - `closedWeekdays` / `businessHours` / `menus`: `config.js` と同じ内容に揃える

### 3. ウェブアプリとしてデプロイする

1. 画面右上の「デプロイ」→「新しいデプロイ」
2. 種類の選択で「ウェブアプリ」を選ぶ
3. 「次のユーザーとして実行」: 自分（＝カレンダーの持ち主のアカウント）
4. 「アクセスできるユーザー」: 全員
5. 「デプロイ」をクリックし、初回は権限の承認（カレンダー・Gmail・スプレッドシートへのアクセス許可）を行う
6. 発行された **ウェブアプリのURL**（`https://script.google.com/macros/s/xxxx/exec`）をコピー

### 4. フロントエンドに接続する

`config.js` を開き、`API_URL` に手順3で取得したURLを設定する:

```js
API_URL: "https://script.google.com/macros/s/xxxxxxxxxx/exec",
```

保存して `index.html` を開き直すとデモバナーが消え、実際にGoogleカレンダーへ予約が登録されるようになります。

### 5. コード修正後の再デプロイ

`Code.gs` を修正した場合は、「デプロイ」→「デプロイを管理」→ 該当のデプロイの鉛筆アイコン →「バージョン」で「新バージョン」を選んで再デプロイしないと反映されません。

## カスタマイズ

- 店舗名・メニュー・スタッフ・営業時間・定休日は `config.js`（フロント）と `gas/Code.gs` の `CONFIG`（バックエンド）の**両方**を同じ内容に揃えてください
- 予約枠の刻み幅は `slotIntervalMin`（分）で変更できます
- 配色は `style.css` 冒頭の `:root` 内のCSS変数（`--ac` など）で調整できます
